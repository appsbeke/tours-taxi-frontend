## 🚀 Deployment Instructions

### Quick Deploy (Recommended)

1. **Go to Vercel:**
   - Visit: https://vercel.com/dashboard
   - Click "Add New Project"

2. **Import Repository:**
   - Select: `appsbeke/tours-taxi-frontend`
   - Click "Import"

3. **Configure Environment:**
   ```env
   NEXT_PUBLIC_API_URL=https://tours-taxi-backend.onrender.com
   NEXT_PUBLIC_SOCKET_URL=https://tours-taxi-backend.onrender.com
   NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_key_here
   NEXT_PUBLIC_STRIPE_PUBLIC_KEY=your_key_here
   ```

4. **Deploy:**
   - Click "Deploy"
   - Wait 2-3 minutes
   - Get your live URL!

### CLI Deploy (Alternative)

```bash
# Navigate to project
cd /root/.openclaw/workspace/tours-taxi-frontend

# Login to Vercel
vercel login
# Follow the authentication flow

# Deploy to production
vercel --prod
```

## 📚 API Integration

### Endpoints Used
- `POST /api/auth/login` - User login
- `POST /api/auth/register` - User registration
- `GET /api/auth/me` - Current user
- `GET /api/rides` - List rides
- `POST /api/rides` - Create ride
- `PATCH /api/rides/:id/cancel` - Cancel ride
- `GET /api/tours` - List tours
- `GET /api/tours/:id` - Tour details
- `POST /api/tour-bookings` - Book tour
- `GET /api/tour-bookings/my-bookings` - User bookings
